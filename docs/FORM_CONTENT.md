# Project Form Content

### Project Title

DevOps-Driven Learning Management System (LMS)

### Deliverables / Tasks

1. Develop a professional, responsive full-stack web application using Next.js and Express
2. Implement secure user authentication and role-based authorisation for students and instructors
3. Design and integrate a database (NeoneDB/Postgress) for users, courses, lessons, and progress tracking
4. Build an interactive dashboard for student progress tracking and course management
5. Use Git and GitHub for structured version control with branching strategy
6. Apply Agile/Scrum development practices across multiple sprints
7. Containerise the Express backend and Next.js frontend using Docker and docker-compose
8. Implement an automated CI/CD pipeline using GitHub Actions to automate linting and test runs
9. Deploy the frontend to Vercel and the backend to Render cloud platform
10. Produce complete software process lifecycle documentation
